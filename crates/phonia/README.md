# phonia

Phonia is an open-source toolkit for phonetic research. This crate reserves the
`phonia` name for the project; it carries no library API.

The application lives at <https://phonia.app>. The toolkit's library crates are
published under the `phx-` prefix — for example [`phx-annot`] and
[`phx-textgrid`]. Source is at <https://github.com/mkpoli/phonia>.
